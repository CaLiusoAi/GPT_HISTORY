## STEP 2: Zenodo (Permanent DOI)

**Why:** CERN-backed permanent archive. Even if GitHub disappears, Zenodo doesn't. Gives you a DOI ‚Äî a permanent citation address.

1. Go to https://zenodo.org
2. Create account (free)
3. Click "New Upload"
4. Upload the entire archive as a ZIP
5. Use the metadata in `ZENODO_METADATA.json`
6. Publish

**Result:** A DOI like `10.5281/zenodo.XXXXXXX` ‚Äî permanently citable in any academic context.

**Bonus:** Connect your GitHub repo to Zenodo ‚Äî it will automatically archive every release.
